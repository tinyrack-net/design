## Contract

`TRBrand` renders a linked logo followed by an optional title and any trailing content, such as a version badge. Pass the logo as the `logo` node and supply a destination through `href` or a framework link with `render`.

## Examples

Compose `TRBrand` inside a shell header or sidebar. The `render` prop accepts a router link element so the logo participates in client-side navigation, and trailing children render after the title for badges or environment labels.

## API

### `TRBrand`

| Prop | Type | Default | Purpose |
| --- | --- | --- | --- |
| `logo` | `ReactNode` | Required | Logo mark or image rendered inside the link. |
| `href` | `string` | `undefined` | Destination for the logo link. |
| `render` | `ReactElement` | Native anchor | Framework link element merged with the logo link. |
| `title` | `ReactNode` | `undefined` | Optional text rendered next to the logo. |
| `titleClassName` | `string` | `undefined` | Class applied to the title element. |
| `linkClassName` | `string` | `undefined` | Class applied to the logo link. |

`TRBrand` also forwards native `div` props, `className`, `style`, events, data attributes, and a React 19 `ref`. Trailing `children` render after the title.

## Styling

Import `@tinyrack/ui/components/brand.css` after core tokens. `.tr-brand` is a flex container, `.tr-brand-link` wraps the logo, and `.tr-brand-title` styles the title text.
